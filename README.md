# Noted... 

**Noted** is a minimalist high fidelity visual workspace for note-taking, managing content built with React. The app is designed to offer a distraction-free, intuitive interface for jotting down notes, saving different media(such as GIFs, PDFs, etc) with focus on interactive experiences. The app features an infinite canvas(tldraw) to place different formats of content and curate with them. 

## Features 🚀

- Infinite canvas(tldraw whiteboard) to place all your content.
- Responsive design, optimized for all screen sizes.
- Intuitive UI with major focus on interactivity & responsiveness.
- Simple, easy-to-use interface for adding and managing your content.

## Tech Stack 🛠️

- **React**: A JavaScript library for building user interfaces.
- **Vite**: A fast build tool for modern web projects.
- **Shadcn**: A set of beautifully designed, composable & customizable components.
